About Our Calculator Web App

This calculator web application is built using Python Flask, Jinja2, HTML, CSS, and Bootstrap. It allows users to perform fast and accurate basic arithmetic operations including addition, subtraction, multiplication, and division.

Why This App?

Our mission is to provide a lightweight, responsive, and user-friendly calculator accessible on both desktop and mobile devices. With a clean interface and easy navigation, you can calculate instantly without unnecessary distractions.

Features

  • ✔️ Perform basic operations (Add, Subtract, Multiply, Divide)
  • ✔️ Login and Signup functionality
  • ✔️ Calculator history to review past calculations
  • ✔️ Responsive Bootstrap design for all devices

Navigation

Use the navigation bar above to explore:

  • Home – Start calculations quickly
  • About – Learn more about this project
  • History – View your calculation records
  • Login / Signup – Manage your account
